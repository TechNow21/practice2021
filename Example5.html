Example 5

Lab 5

Brief Instructions:

Creating Poster

The objective of this project was to use pictures and text to create a poster that would get people's attention. Here, I use the tree picture as the background and used the jack-0-lanterns in the title of the poster.

I chose to do it this way to show how different images could be used in one.

What I did:

1. Created a poster by selecting File - New. Size = 18 x 24; Resolution = 72.

2. Added background photo (trees).

3. Added Text 1 (Halloween in the Dark). Added a Clipping Mask and Drop Shadow: angle 42º; Opacity 50%; Distance 37 px; Spread 28%; Size 49 px. Character Panel: Tracking 25; Warp Text = Arch, horizontal.

4. Added Text 2 (Trick or Treat). Modified style and size. In Character Panel: Warp Text = Wave. Added Filters: Liquify and modified the T’s. Added Filter - Texture - Texturized.

5. Added Text 3 (Spooky Party and Midnight | October 31). Added Text Path - Adjustments panel - Hue/Saturation - Lightness = -29. Added Filter - Liquify.

6. Added Shape 1 to Text 2, Cat. Rotate = -30º. Added Fill = Grayscale, no Stroke. Added Filter - Filter Gallery - Film Grain. Added another Filter - Stylize - Wind.

7. Added Shape 2 to Text 2, Flower. Added Filter - Liquify. Fill - Dark = Yellow, Pure = Yellow. Added another Filter - Ocean Ripple.

8. Saved and Exported image for web friendly version.
