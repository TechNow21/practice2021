Example 1

Lab 1 - Part A

Brief Instructions:

Opening, Changing Size, Cropping and Saving

The idea in this project was to learn to open, resize, crop, and save pictures. In this way a modified picture was created to the desired look. Here, I created a close up of the butterflies and flowers.

I chose to do it this way because I wanted to be able to crop pictures the way I wanted.

What I did:

1. Opened image in Photoshop.

2. Used Image - Image Size and reduced width and height to 41% original size to get largest edge under 800 pixels (790 pixels x 525 pixels).

3. Cropped out some of the areas to the around the butterfly and yellow flowers the resulting image is 790 pixels x 525 pixels.

4. Exported image for web friendly version.
