Example 5

Lab 6

Brief Instructions:

Creating a Painting

The objective in this project was to create a picture from scratch and practice painting it using Photoshop tools. I made an sketch of some trees in a field with the blue sky and sun; then, painted the details.

I chose to do it this way because I wanted to learn how to draw and paint using digital tools.

What I did:

1. Created a paining project - New. Size = 1500 x 1000 pixels; Resolution = 300 px/in.

2. Layer 1: Background

3. Layer 2: Sketch – Drew basic outlines for paining

4. Layer 3: Grass (A) Used ‘round fan brush’ to paint background green. (B) Used ‘Kyle’s concept brush’. (C) Mixed some background colors with other green color. (D) Used ‘grass highlight brush’ – preset from Lesson 10 files.

5. Layer 4: Sky (A)Used ‘sky brush’ – preset from Lesson 10 files. (B) Used ‘Kyle’s concept brush. (C) Mixed blue colors in the background. (D) Used ‘clouds brush’ – preset from Lesson 10 files.

6. Layer 5: Trees (A) Used modified brush ‘spatter ktw 3’- size 27 px; color #7a4f17. (B) Used textured brush ‘kyle’s screentones 38’ for the trunks.

7. Layer 6: Tree Flowers/leaves (A) Used ‘kyle’s spatter brush’ to add some flowers and leaves. (B) Used ‘leaves brush’ – downloaded brush. (C) Used ‘flower brush’ – downloaded brush.

8. Layer 7: Sun (A) Used ‘hard round’ brush to color the sun yellow. (B) Added another yellow color to mix with previous color. (C) Added some glow to make it shine:

  1. Used ‘pen tool’ to select image
  2. Duplicated layer
  3. Converted to Smart object
  4. Made a folder to Group layers
  5. Duplicated layer
  6. Changed blending mode to ‘screen’
  7. Used: Filter - Blur - Gaussian Blur - Radius 5px
  8. Duplicated layer
  9. Opened ‘smart filter’, Radius 150 px
  10. Duplicated layer; Radius 300 px
  11. Added Hue/Saturation Adjustment layer
  12. Created clipping mask
  13. Colorized and changed saturation levels
